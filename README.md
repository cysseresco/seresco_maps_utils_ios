### Installation
Installation
--------

#### CocoaPods

```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '12.0'
use_frameworks!

target 'MyApp' do
  pod 'SerescoMapsUtils', '~> 1.0'
end
```
